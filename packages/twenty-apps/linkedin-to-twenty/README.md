# Linkedin to Twenty 👋

A tool to sync LinkedIn profiles to Twenty

## 🌟 What it does

This app provides a tool to sync LinkedIn profiles to Twenty:
- Syncs LinkedIn profiles to Twenty
- Creates a new user in Twenty for each LinkedIn profile
